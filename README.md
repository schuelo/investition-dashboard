# Investition Dashboard V27.0 – Professional Dark

V27 ist ein vollständiges visuelles Redesign auf Basis von V26. Sämtliche bestehenden Funktionen, Supabase-Anbindungen, News Intelligence, Depotpositionen, Trade-Pläne und Edge Functions bleiben erhalten.

## Neu in V27

- Professional-Dark-Designsystem
- feste Desktop-Seitenleiste
- iPhone-optimierte Bottom-Navigation
- klarere Kennzahlen- und Kartenhierarchie
- aufgewertete News- und Portfolioansichten
- optimierte Touchflächen und mobile Abstände
- zurückhaltende Grün-/Rot-Signalisierung; Türkis als primärer Akzent

## Installation

1. Bestehendes Dashboard exportieren.
2. Alle Dateien dieses Pakets in das Hauptverzeichnis des GitHub-Repositories laden und vorhandene Dateien ersetzen.
3. Commit ausführen.
4. `https://schuelo.github.io/investition-dashboard/reset.html?v=27.0` öffnen.
5. Zurücksetzen und anschließend `https://schuelo.github.io/investition-dashboard/?v=27.0` öffnen.

Für V27 sind keine neuen Supabase-Tabellen oder Edge Functions erforderlich. Die V26-News-Backend-Dateien bleiben unverändert in Betrieb.
